# iNotebook - Your notes on the cloud
- Save Your Notes secured on the cloud
- live site: https://my-inotebook.netlify.app/

## Made with MERN STACK

## Features
- Create Account
- Sigin
- Logout
- Private Note
- Get your details after signup
- Delete your account

## Notes Features
- create note
- read note
- update note
- delete note
